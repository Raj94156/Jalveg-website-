## 🌐 Website Sections

1. **Hero** — Animated intro with product image and CTA buttons
2. **Stats** — Key numbers (5 sizes, 100% pure, 7 quality checks, 24×7)
3. **Features** — 6 feature cards (purification, minerals, hygiene, etc.)
4. **Products** — 5 bottle sizes from 250ml to 20 Litre
5. **Factory** — Photo gallery of production facility
6. **About** — Company story + both owners with photos
7. **Certifications** — FSSAI, GST, BIS, Lab Tested, BPA Free + product brochure
8. **Testimonials** — 6 customer reviews
9. **Contact** — Enquiry form + company details
10. **Footer** — Links, address, license info



## ✨ Features

- Pure HTML + CSS + JavaScript (no frameworks, no build tools)
- Fully responsive (mobile, tablet, desktop)
- Purple theme with animated backgrounds
- Floating bubbles animation
- Scroll reveal animations
- Animated number counters
- Smooth scroll navigation
- WhatsApp direct enquiry button
- Form validation (name: letters only, phone: numbers only)
- Golden framed product brochure display
- Glassmorphism cards


## 🚀 How to Run Locally

Simply open `index.html` in any browser — no server needed.

Or run a local server:
```bash
# Python (recommended)
python -m http.server 5173
# Then open http://localhost:5173
```


## 🌍 How to Host

### Netlify Drop (Free — Easiest)
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag the `jalveg-website` folder onto the page
3. Get a live link instantly

### Hostinger
1. Buy a plan at [hostinger.com](https://hostinger.com)
2. Go to hPanel → File Manager → `public_html`
3. Upload and extract the ZIP of this folder
4. Make sure `index.html` is directly inside `public_html/`

### GitHub Pages (Free)
```bash
git init
git add .
git commit -m "Jalveg website"
git branch -M main
git remote add origin https://github.com/yourusername/jalveg-website.git
git push -u origin main
```
Then go to GitHub repo → Settings → Pages → Deploy from `main` branch
